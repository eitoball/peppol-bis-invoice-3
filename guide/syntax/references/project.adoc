
[[projectref]]
= 計画参照

計画参照は任意であり、請求書では、要素 `cac:ProjectReference/cbc:ID` で送信されます。クレジットノートでは、この要素は存在せず、`cac:AdditionalDocumentReference[cbc:DocumentTypeCode='50']/cbc:ID` で送信されます。

NOTE:: 計画参照を送信する場合、`cac:AdditionalDocumentReference` 要素では `cbc:ID` と `cbc:DocumentTypeCode` のみが利用できます。
