= EN16931 の制限

この Core Invoice Usage Specification (CIUS) で行われている {EN16931} の制限事項の一覧を以下に示します。これらのルールに加えて、売り手の国や付加価値税（VAT）の国によっては、さらに制限が設けられている場合もあります。 国ごとのルールの一覧は <<national_rules>> にあります。

すべての制限は、 {EN16931} の第7.3.2章に記載されているように、 {CUIS} で許容される仕様の範囲内です。
