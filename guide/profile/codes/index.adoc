
= 請求書型コード（BT-3）の一覧 (BT-3)


以下の節では、PEPPOL BIS Billing で使用される請求書型コードの制限されたセット（BT-3）の概要を説明します。

表は、各コードの名称と説明、および、受取人がこのコードのための別個のプロセス／ワークフローを持っていない場合に、この請求書タイプがどのように処理できるかを示す列「同義コード」を示しており、このプロファイルで有効なすべての請求書型はビジネス請求書（３８０）として、すべてのクレジットノート型はビジネスクレジットノート（３８１）として処理することができます。一部の受信者は、これらの型のいくつかについて別々のプロセスやワークフローを持っているかもしれず、これらのプロセスを使用することができます。

まは、表は、異なるコードに対してUBL構文で使用されるメッセージ型に関する情報を提供します。
