= 商品情報

== 商品識別子

明細行内では、売り手の商品識別子、買い手の商品識別子、標準商品識別子を提供することができます。売り手と買い手の商品識別子については、スキーマ属性は使用されず、標準商品識別子には、 `schemeID` が必須であり、ISO 6523 ICD の一覧からのものである必要があります。

== 商品分類
