= 数量の単位

請求書の単位は、UNECE勧告第20号（バージョン11e）のコードと、前にXを付けたUNECE勧告第21号のコードを使用することができます。
コードリストの詳細については、 <<uom>> を参照して下さい。

.勧告第20号からの数量の単位の例
[cols="1,5", options="header"]
|===
|コード
|数量の単位

|H87
|個

|KGM
|キログラム

|MTR
|メーター

|LTR
|リッター

|MTK
|平方メーター

|MTQ
|立方メーター

|KTM
|キロメーター

|TNE
|トン（メートルトン）

|KWH
|時間当たりのキロワット

|DAY
|日

|HUR
|時間

|MIN
|分
|===

.前にXを付けた勧告第21号からの数量の例
[cols="1,5", options="header"]
|===
|コード
|数量の単位

|XBG
|袋

|XBX
|箱

|XCT
|カートン

|XCY
|シリンダー

|XBA
|バーレル

|XPK
|パッケージ

|XPX
|パレット

|XRL
|巻

|XSA
|袋

|XST
|枚
|===
