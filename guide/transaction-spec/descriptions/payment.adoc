
= 支払方法情報

== 銀行口座振替

銀行口座振替の場合は、決済口座識別子(BT-84)が必須となります。

すべての許可されたコードについては、 <<payment-means>> を参照してください。銀行口座振替による支払いのためのコードの例は次のとおりです。

* 30 - 銀行口座振替
* 58 - SEPA 銀行口座振替

include::../../syntax/credit-payment.adoc[]

== カード支払い

購入者がクレジットカードやデビットカードなどの支払いカードを使用して支払うことを選択した場合、請求書にはプライマリアカウント番号（PAN）の情報が記載されている必要があります。

許可されているすべてのコードについては、 <<payment-means>> を参照してください。カード払いのコードの例は以下の通りです。

* 48 - 銀行カード
* 54 - クレジットカード
* 55 - デビットカード

include::../../syntax/card-payment.adoc[]
