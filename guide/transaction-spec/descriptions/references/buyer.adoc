= 購入者参照

購入者の参照番号は、 **条件付き** となります。請求書には、購入者参照、または、注文参照のいずれかを記載する必要があります（ <<orderref>>> を参照）。

この要素は、製品やサービスを注文した人の参照に使用されます。例としては、注文者の名前、従業員番号、または、購入者自身や部署、もしくは、グループを識別するコードなどがあります。購入者の参照は、受信者の内部ルーティングに使用されることが多いので、受信者の必要性に応じてこの要素に正しい値を記入することが重要です。

購入者参照も注文への参照も顧客によって提供されない場合、納入業者が知っていれば、発注者、または、顧客のために任命された人の名前を購入者参照で提供することができます。


IMPORTANT: 参照が顧客によって提供された場合，正しい要素が，提供された参照を含んでいる必要があります。
