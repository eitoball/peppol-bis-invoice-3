
= 参照情報

請求書検証のサポートは、請求書での重要な機能です。請求書は、電子的、または、紙媒体を問わず、関連する既存の文書を調べるのに十分な情報を提供しなければなりません。

IMPORTANT: 参照要素には有効な情報が含まれている必要があります。参照がない場合は、その要素が文書インスタンス内に存在してはいけません。
