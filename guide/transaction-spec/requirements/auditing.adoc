
[[auditingreq, Auditing requirements]]
.監査プロセス要求
[cols="1,10", options="header"]
|===
|識別子
|（それぞれの取引に応じての）要求

|R56
a|以下の点に関する監査プロセスをサポートするのに十分な情報:

*   請求書の識別子;
*   請求書の発行日;
*   説明、金額、数量など取引された製品やサービスの特定;
*   請求書の決済に関連する情報;
*   契約書、発注書、発送通知書などの関連書類と請求書を関連付けるための情報;

|R57
a|請求書での法的名称、および、住所を含む、以下の役割を果たす当事者の識別

*   売主（売主の商号を含む）;
*   買手
*   （買い手と異なる場合）配送先;
*   （売主と異なる場合）受取人;
*   供給者の税務担当者;
|===
