
= 明細行の計算

[[price-calc]]
== 請求明細行の正味金額 (BT-146)

総額と割引額が存在する場合、明細行の正味金額は、明細の総額から割引額を差し引いたものと同じである必要があります。

計算式:

====
stem:["明細行の正味金額" = "明細の総額 (BT-148)" - "明細の割引額 (BT-147)"]
====

== 請求明細総額 (BT-131)

請求明細総額 (BT-131) は、その名の通り、付加価値税（VAT）抜きの明細行での引当金と手数料を含む合計額です。

請求明細総額の計算式は以下になります:

====
stem:["請求明細総額" = (("明細合計額 (BT-146)" div "商品価格基準数量 (BT-149)")] +
stem:[times ("数量 (BT-129)")] +
stem:[+ "明細手数料 (BT-141)" - "明細引当金 (BT-136)"]
====

IMPORTANT: 請求明細合計は、小数点以下2桁に四捨五入する必要があるため、計算の異なる部分は個別に四捨五入する必要がありますのでご注意ください。 +
つまり、
stem:["請求明細総額" = (("明細合計額 (BT-146)" div "商品価格基準数量 (BT-149)") times ("数量 (BT-129)")] の結果は、
小数点以下2桁に四捨五入しなければならず、手当・請求額も別途四捨五入しています。
