
= 付加価値税（VAT）計算

明細行の付加価値税（VAT）情報か、文書の割引、もしくは、手数料のどちらかに記載されている付加価値税（VAT）コードと割合の異なる組み合わせごとに付加価値税（VAT）の明細を1つ提供する必要があります。

各付加価値税（VAT）カテゴリーコードと割合の明確な組み合わせについては、以下のように計算されます。
====
stem:["付加価値税（VAT）カテゴリーの課税対象額 (BT-116)" = sum("請求明細行の合計 (BT-113)")] +
stem:[+ "文書での手数料 (BT-99)" - "文書での割引額 (BT-93)"]
====

====
stem:["付加価値税（VAT）カテゴリーの税額 (BT-117)" = "付加価値税（VAT）カテゴリーの課税対象額 (BT-116)" times ("付加価値税（VAT）割合 (BT-119)" div 100)]
====

IMPORTANT: 付加価値税（VAT）カテゴリーが「付加価値税（VAT)(対象外」（O）である付加価値税（VAT)明細については、そのカテゴリーの税額は、0 とします。
